This website serves as an online portfolio to showcase my skills, projects, and experiences as a programmer. It highlights my academic background in Computer Science, my project involvement during my diploma studies, and passion related to innovate creative stuff. The website also provides a way for potential collaborators, employers, or anyone interested to get in touch with me directly.

Tehnologies Used

HTML5: For structuring the content of the website.

CSS3: For styling and layout of the website. A custom style.css is used for additional styling tweaks.

Bootstrap 5: A front-end framework used for responsive design and pre-built components, such as the navbar and cards.

JavaScript: For dynamic content like form submission handling and interactivity (Bootstrap's JavaScript components and custom scripts).

Bootstrap Icons: For the usage of vector icons like the envelope, phone, and LinkedIn icons.

Animate.css: For adding smooth animations and transitions to elements when they come into view.

WOW.js: For triggering animations when the user scrolls through different sections.

Formspree: For handling form submissions without needing a backend server.

Special Functionalities

Modal Pop-ups: Each project listed in the "Projects" section has a modal pop-up that reveals more detailed information about the project and includes embedded YouTube videos demonstrating each project.

Email & Contact Information: The "About Me" section includes clickable icons for email, phone, and LinkedIn, allowing easy contact with me.

Form Submission: The Contact section includes a form that submits messages to my email via Formspree, simplifying communication.
